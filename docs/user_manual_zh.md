# AI 任务助手 — 用户使用手册（简版）

本文档面向**最终用户**，介绍常见能力与使用方式。技术细节与接口定义见 [`docs/api/openapi.yaml`](../api/openapi.yaml)。

## 1. 登录与账号

- **匿名昵称登录**：输入显示名即可使用；若该名称尚未注册，会创建新账号；若已存在，则直接登录该账号。
- **邮箱**（可选）：新用户可在登录时填写邮箱，便于找回或资料展示；**同一邮箱不能绑定多个账号**。
- **令牌**：登录成功后客户端会保存 **JWT**，后续请求放在 `Authorization: Bearer <token>`。开发环境也可使用预置的测试用户 id（由部署方提供）。

## 2. 对话与多会话

- **会话列表**：侧边栏或会话列表按**最近更新时间**排序；可**新建会话**。
- **进入会话**：选择会话后加载历史消息；支持**分页**向上翻阅更早记录。
- **重命名**：用户修改会话标题后，系统会标记为**用户命名**，不再被自动标题覆盖。
- **删除会话**：删除后该会话下消息一并清除（以产品实际交互为准）。

## 3. 发送消息（流式回复）

- 每条发送须属于**当前会话**（后端校验 `session_id`）。
- 回复以**流式**展示：逐字或逐段出现；若网络中断可重试发送或刷新页面。
- **引用与搜索**：当助手使用「工作区文件记忆」或「联网搜索」时，界面可能显示**引用角标**或**来源悬停信息**（与具体前端实现一致）。

## 4. 个人资料

- 可修改**显示名**（全站唯一，与他人重复会提示失败）。
- **邮箱**可留空或清空；修改邮箱时若与他人重复会提示失败。
- 可设置 **AI 助手昵称**（对你的称呼方式）。
- **偏好**（若有）：以 JSON 对象形式保存，用于个性化提示词（具体字段见产品说明）。

## 5. 任务管理

- 支持创建、查看、编辑、删除**任务**；部分任务支持结构化 **详情**（子任务等，视产品界面而定）。
- 可按**状态**、**项目**筛选任务列表。

## 6. 工作区文件

- **上传**：小文件可直接上传；大文件使用**分片上传**（前端展示进度）。
- **文件夹路径**：列表支持按路径前缀浏览，便于整理资料。
- **标签与语义类型**：可为文件打标签或设置类型，便于助手在工作区工具中检索。
- **处理状态**：文件上传后可能在后台**解析与建索引**；失败时可使用「重试处理」（若产品提供入口）。状态含义：`0` 处理中，`1` 成功，`-1` 失败。

## 7. 常见问题

| 现象 | 可能原因与处理 |
|------|----------------|
| 登录后仍提示未授权 | 令牌过期或无效；重新登录。开发环境确认后端已创建对应测试用户。 |
| 发送消息提示会话不存在 | 会话已被删除或未同步；返回列表重新进入或新建会话。 |
| 修改昵称失败 | 该昵称已被他人使用；换一个名称。 |
| 搜索/记忆无结果 | 未配置文件存储或向量服务；或文件仍在处理中。联系管理员检查部署。 |
| 流式回复卡顿或中断 | 模型或网络波动；可重试。长时间无 `done` 事件可刷新页面。 |

## 8. 隐私与安全建议

- 勿在公共设备上勾选「长期记住登录」除非信任该设备。
- 工作区文件可能进入「记忆检索」范围，请勿上传敏感或违规内容。
